# Oath and Bone — Seamless Biome Texture Prompt Bank
# 42 prompts: 7 biomes × 6 variants, all --tile for seamless edge-matching
# Session: April 21, 2026
#
# Design principle: subtle micro-variation per variant. NO dominant focal features
# (no lone rocks, trees, wells, arches). Each variant reads as "this stretch of
# grassland/forest/etc has slightly different local texture." At runtime the game
# shuffles variants per hex; with 6 variants + 4-rotation + 2-mirror = 48 effective
# tiles per biome, visible repetition becomes statistically invisible across any
# reasonable play area.
#
# STYLE ANCHOR (shared across all 42):
#   painterly hand-painted oil-painted style, Vagrant Story Unicorn Overlord
#   Final Fantasy Tactics PS1 game art aesthetic, muted kingdom-grey palette with
#   [BIOME ACCENT] accents, dark atmospheric lighting, top-down orthographic view
#   of flat ground, tactical RPG environment art, no characters, no UI, no text,
#   no dominant features, continuous uniform ground cover with subtle local variation
#   --tile --ar 1:1 --style raw --v 6

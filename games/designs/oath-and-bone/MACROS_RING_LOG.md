## Pipeline note

All 30 files delivered via IndexedDB-ZIP workaround documented in PORTRAITS_TRIO_A_LOG.md. Cloudflare TLS fingerprinting (JA3/JA4) blocks curl; Chrome popup blocker blocks multi-file programmatic downloads. The working chain: load blobs via CORS-anonymous canvas → convert to ArrayBuffers → IndexedDB save → fresh tab → read IDB → hand-roll STORE ZIP → single `a.click()` download → `unzip` + `mv`. ~20 min end-to-end for 30 files.

Alternates `0_1..0_3` remain on the CDN at `cdn.midjourney.com/{jobId}/0_{1,2,3}.webp`. Fetch on demand if a different variant reads better.
